## 2024-05-24 - Consistent Keyboard Focus Indicators
**Learning:** Adding explicit `:focus-visible` styles with dashed white outlines and offsets ensures keyboard users have clear, high-contrast visual feedback when navigating custom interactive elements like game buttons, without impacting mouse users.
**Action:** Always apply `:focus-visible` with a distinct `outline` and `outline-offset` to all custom buttons and links in this design system.
